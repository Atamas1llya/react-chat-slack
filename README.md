# react-chat-slack

## Usage

```
import ReactChatSlack from 'react-chat-slack';

class App extends Component {
  render() {
    return (
      <ReactChatSlack
        token="your_token_here"
        channel_id="slack_channel_id"
        username="user_name_to_display"
        title="Chat title"
      />
    )
  }
}
```
